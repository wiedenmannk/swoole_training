# Setup

Willkommen zur Swoole-Schulung.

Bevor wir mit den ersten Beispielen beginnen, muss eine lauffähige Entwicklungsumgebung eingerichtet werden.

Je nach Betriebssystem stehen unterschiedliche Installationswege zur Verfügung.

## Installationsvarianten

### Linux (empfohlen)

Die empfohlene Umgebung für Swoole.

Enthält die vollständige Installation mit PHP, PECL und der Swoole-Erweiterung.

→ `linux/README.md`

---

### Windows

Für Windows empfehlen wir die Verwendung von Docker.

Eine native Windows-Installation wird in dieser Schulung nicht behandelt.

→ `windows/README.md`

---

### Docker

Docker stellt eine fertige Entwicklungsumgebung bereit und eignet sich besonders für Windows-Systeme oder als Notfalllösung.

→ `docker/README.md`
