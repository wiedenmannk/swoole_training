# Linux

Diese Anleitung beschreibt die Einrichtung einer lokalen Swoole-Entwicklungsumgebung unter Linux.

Die Installation erfolgt Schritt für Schritt.

## Inhalte

### 1. Voraussetzungen

Installation und Prüfung der benötigten Werkzeuge:

- PHP
- PECL
- Compiler (gcc, make)
- PHP-Entwicklungspaket (php-dev)

→ `01-voraussetzungen.md`

---

### 2. Swoole installieren

Installation der Swoole-Erweiterung über PECL.

Während der Installation werden verschiedene Optionen abgefragt. Für diese Schulung verwenden wir eine empfohlene Standardkonfiguration.

→ `02-swoole-installation.md`

---

### 3. Installation prüfen

Überprüfung der Installation und erster Test des Swoole-Servers.

→ `03-installation-pruefen.md`

---

## Ziel

Nach Abschluss dieser Anleitung steht eine lauffähige Swoole-Entwicklungsumgebung zur Verfügung, die für alle weiteren Beispiele dieser Schulung verwendet werden kann.
